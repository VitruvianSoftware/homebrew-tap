# Homebrew Tap for VitruvianSoftware

This repository contains Homebrew formulae for tools built by [VitruvianSoftware](https://github.com/VitruvianSoftware).

## Available Formulae

| Formula | Description |
|---------|-------------|
| `homelab` | Declaratively provision and manage multi-node K8s homelab clusters on macOS |

## Usage

```bash
# Add the tap
brew tap vitruviansoftware/tap

# Install homelab
brew install vitruviansoftware/tap/homelab
```

## Updating

Formulae are automatically updated by [GoReleaser](https://goreleaser.com/) when new releases are published to the [homelab](https://github.com/VitruvianSoftware/homelab) repo.
